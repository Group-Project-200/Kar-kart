This Branch is a rudimentary application of a little car game with a working but unfinished physics engine.
Use 'W' 'A' 'S' and 'D' as well as a dumb implementation of a handbrake when pressing 'space'.
Drifting physics not added. Collisions neither.

RESOLUTION AND DISPLAY SIZE ARE DONE AS PARSERS
Try running 'python3 main.py -h' to see details
